# Vehicle-Analysis-Dashboard
Hii guys,
I have designed an interactive Vehicle Analysis Dashboard in Power BI to provide comprehensive insights into vehicle performance, operational costs, and regional distribution. The dashboard helps users analyze key vehicle metrics and identify trends through dynamic visualizations and filters. <br>

**Key KPIs**
Total Vehicles (3000): Displays the total number of vehicles available in the dataset.
Maintenance Cost (3M): Shows the total maintenance expenditure across all vehicles.
Average Range (374.41): Represents the average driving range of vehicles.
Average Mileage (125.21K): Displays the average mileage achieved by the vehicles.
<br>

**Interactive Filters**
Year Slicer: Allows users to filter data by manufacturing year.<br>
Model Slicer: Enables analysis of specific vehicle models.<br>

**Visualizations Used**
Total Vehicle by Model <br>
Bar chart showing the distribution of vehicles across different models. <br>
Helps identify the most common vehicle models in the dataset.<br>
Count of Make by Region<br>
Column chart displaying vehicle counts across regions such as Asia, Europe, Australia, and North America.<br>
Useful for understanding regional vehicle distribution.<br>
Count of Model by Average Speed (km/h)<br>
Visual representation of vehicle models based on their average speed.
Helps analyze performance variations among different vehicles.
Insurance Cost vs Maintenance Cost by Model
Comparative line chart showing maintenance and insurance expenses for each vehicle model.
Assists in identifying high-cost vehicles and evaluating overall ownership expenses.
Features
Dynamic filtering using slicers.
Interactive KPI cards for quick insights.
Comparative cost analysis.
Regional and model-wise performance tracking.
User-friendly dark theme dashboard design for better visualization.
Business Value

This dashboard enables stakeholders to monitor vehicle performance, compare operational costs, identify high-maintenance models, and make informed decisions based on data-driven insights. It provides a centralized view of critical vehicle metrics, improving analysis efficiency and reporting accuracy.
